# Advisor Template

Use this template when adding a new advisor.

## Required Files

- `persona-spec.md` or `persona-spec.zh-CN.md`
- `ingestion-policy.md`
- `system-prompt.md`
- `source-registry.yaml`

## Minimum Standard

- At least 8 high-signal sources
- At least 5 recurring traits mapped to evidence
- At least 3 blind spots or contradictions
- A clear statement that this is an inspired advisor, not the real person

## Suggested Fields for Each Source

- `id`
- `title`
- `category`
- `source_type`
- `authority`
- `url`
- `published_at`
- `why_it_matters`
- `reuse_policy`
- `status`
